Navigable plot objects

All lineage plot elements (node / link / node label / link label) have object with contextual information. For example, node object stores not only the input data that was used to create it, but it also keeps a reference to its DOM object, parent and children objects, ingoing link object and the node's x and y position. Node and link objects are injected as parameters in node/link click/mouse over/mouse out events. This makes it easy to create interactions with the plot. For example, upon a node click, the whole path leading through all ancestor nodes can be highlighted by traversing all the ancestor nodes iteratively. Most of the parameters available in the objects are listed below.

Node:
  • data (object) The original node object as specified by user in plot-data data input.
  • DOMElement (DOM) Node's DOM element.
  • parent (object) Parent node's object.
  • children (array) Array of children node objects.
  • inLink (object) Ingoing link's object.
  • label (object) Node's label object.
  • bbox (object) Bounding rectangle for label collision detection purposes.
  • depth (number) Generation index of the node.
  • x (number) Current position x of the node.
  • y (number) Current position y of the node.
  • _x (number) Original position x of the node (after creating d3 tree layout).
  • _y (number) Current position y of the node (after creating d3 tree layout).
Link:
  • DOMElement (DOM) Links's DOM element (SVG's path).
  • sourceNode (object) Object of the node that the link originated from.
  • targetNode (object) Object of the node that the link goes to.
  • label (object) Link's label object.
Node's label:
  • DOMElement (DOM) Node label's DOM element (SVG's text).
  • text (string) Text of the label.
  • node (object) Corresponding node's object.
  • currentLabelPos (string) Determines the current position of the label with reference to the node. One of objects from layout.nodeLabelPositions array.
  • x (number) Current position x of the label (anchored according to currentLabelPos).
  • y (number) Current position y of the label.
  • dy (number) Vertical offset of the label.
  • width (number) Width of the label.
  • height (number) Height of the label.
  • bbox (object) Bounding rectangle for label collision detection purposes.
  • isColliding (boolean) Specifies whether collisions has been detected for the label.
Link's label:
  • DOMElement (DOM) Link label's DOM element (SVG's text).
  • text (string) Text of the label.
  • link (object) Corresponding link's object.
  • x (number) Current position x of the label (anchored at the center).
  • y (number) Current position y of the label.
  • dy (number) Vertical offset of the label.
  • width (number) Width of the label.
  • height (number) Height of the label.
  • bbox (object) Bounding rectangle for label collision detection purposes.
  • isColliding (boolean) Specifies whether collisions has been detected for the label.
