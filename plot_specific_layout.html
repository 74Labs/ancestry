Plot-specific layout

Lineage plot

  • axis (object)
    • x (object)
      • title (string || default: "") The title of the (horizontal) axis.
      • showAxisLine (true | false || default: false) Determines whether the axis horizontal line is drawn.
      • showGrid (true | false || default: true) Determines whether the grid connected with the axis is drawn.
      • showTickText (true | false || default: true) Determines whether the labels of the axis ticks is visible.
  • brush (object) The general layout of brush can be extended by plot specific properties:
    • axis (object) Identical to plot-specific layout.axis.
  • minViewportWidth (object) If standard lineage plot, set width of a single generation to the number of pixels equal to generationWidth. If time lineage plot, ensure that the time interval in seconds specified by timeIntervalInSeconds has the number of pixels equal to timeIntervalInPixels.
    • generationWidth (number || default: 0)
    • timeIntervalInSeconds (number || default: 2592000 (month))
    • timeIntervalInPixels (number || default: 0)

Lineage-scatter plot

  • axis (object)
    • x (object)
      • title (string || default: "") The title of the (horizontal) axis.
      • format (string || default: "g") Format of the tick labels of the axis according to d3-format.
      • showAxisLine (true | false || default: true) Determines whether the axis horizontal line is drawn.
      • showGrid (true | false || default: true) Determines whether the vertical grid connected with the axis is drawn.
      • showTickText (true | false || default: true) Determines whether the labels of the axis ticks is visible.
    • y (object)
      • title (string || default: "") The title of the (vertical) axis.
      • format (string || default: "g") Format of the tick labels of the axis according to d3-format.
      • showAxisLine (true | false || default: true) Determines whether the axis vertical line is drawn.
      • showGrid (true | false || default: true) Determines whether the horizontal grid connected with the axis is drawn.
      • showTickText (true | false || default: true) Determines whether the labels of the axis ticks is visible.
  • brush (object) The general layout of brush can be extended by plot specific properties:
    • axis (object) Identical to plot-specific layout.axis.
  • link (object) Additional link properties defined for lineage-scatter plot:
    • stroke-dasharray (string || default: '4 4') Dashed line specification. Check here for the appropriate format.

Radial-lineage plot

  • sameLevel ('roots' | 'leaves' | 'both' || default: 'both') Determines whether 'roots' should be places at the same level, or 'leaves', or 'both'.

Phylogenetic-tree plot

  • showBranchLength (true | false || default: true) If true, branches have lengths proportional to the length properties specified in the input data. If false, the length information is ignored and the nodes are spread as much as possible.
